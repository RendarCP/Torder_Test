# Torder_TEST(조성욱)

## 실행방법

- npm install
- npm start
- 라이브러리 사용하여 버그가 발생시 새로고침 해주시면 됩니다.

## 라이브러리

1. redux, react-redux
2. react-router
3. react-scrollspy-nav
